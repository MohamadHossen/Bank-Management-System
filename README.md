# Simple Bank Management System

برنامج بسيط لإدارة حسابات العملاء باستخدام لغة ++C.  
يوفّر إمكانية إضافة العملاء، تعديل بياناتهم، حذفهم، البحث عنهم، وإجراء عمليات الإيداع والسحب، مع حفظ البيانات داخل ملف نصي.

## Features
- عرض قائمة العملاء كاملة.
- إضافة عميل جديد مع التحقق من عدم تكرار رقم الحساب.
- تحديث بيانات أي عميل.
- حذف عميل بعد التأكيد.
- البحث عن عميل باستخدام رقم الحساب.
- تنفيذ عمليات الإيداع والسحب مع التحقق من الرصيد.
- عرض إجمالي أرصدة جميع العملاء.
- تخزين البيانات في ملف Clients.txt.

## How It Works
- يتم حفظ بيانات كل عميل في ملف نصي باستخدام الفاصل #//#.
- عند تشغيل البرنامج تظهر قائمة رئيسية تحتوي على جميع الخيارات.
- كل عملية (إضافة، حذف، إيداع...) تتم عبر دوال منفصلة لسهولة التنظيم.

## How to Run
1. افتح المشروع في أي بيئة تطوير ++C.
2. تأكد من وجود ملف Clients.txt بجانب ملف التنفيذ.
3. شغّل البرنامج من خلال القائمة الرئيسية.

## Notes
- الكود مناسب للتعلم والتدريب على التعامل مع الملفات والدوال.
- يمكن تطوير المشروع لاحقًا لإضافة ميزات جديدة مثل تسجيل الدخول أو واجهة رسومية.
