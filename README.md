
🏦 نظام إدارة البنك

مشروع بلغة C++

برنامج بسيط وقوي لإدارة حسابات العملاء في البنك. تم بناء هذا المشروع بلغة C++ ويتعامل مع ملف نصي لحفظ البيانات بشكل دائم.

---

✨ المميزات

· 📋 عرض كل العملاء – عرض جميع العملاء في جدول مرتب.
· ➕ إضافة عميل جديد – إضافة عميل برقم حساب فريد.
· ❌ حذف عميل – حذف عميل بعد التأكيد.
· ✏️ تعديل بيانات عميل – تعديل معلومات عميل موجود.
· 🔍 بحث عن عميل – البحث برقم الحساب.
· 💸 قائمة المعاملات المالية
  · 💰 إيداع
  · 💳 سحب
  · 📊 إجمالي الأرصدة
· 🔐 حفظ البيانات – كل البيانات تُحفظ في ملف Clients.txt.
· 🧠 قوائم سهلة – قوائم رئيسية وفرعية سهلة الاستخدام.

---

🛠️ التقنيات المستخدمة

· لغة البرمجة: C++
· تخزين البيانات: ملف نصي (Clients.txt)
· المفاهيم: التعامل مع الملفات، الهياكل (Structs)، المصفوفات (Vectors)، الدوال، القوائم (Enums)

---

🚀 طريقة التشغيل

1. حمل المشروع:
   ```bash
   git clone https://github.com/username/Bank-Management-System
   ```
2. افتح المشروع في أي محرر C++.
3. شغل ملف Bank Management.cpp.
4. اتبع القوائم على الشاشة.

---

📌 ملاحظات مهمة

· رقم الحساب يجب أن يكون فريداً لكل عميل.
· لا يمكن السحب أكثر من الرصيد الموجود.
· صيغة حفظ البيانات:
    رقم الحساب#//#الرقم السري#//#الاسم#//#الهاتف#//#الرصيد
